# Cloudflare Workers Utilities

A set of utility functions for Cloudflare Workers.

## Install

```bash
npm install cfw-utils
```

```bash
pnpm add cfw-utils
```

```bash
yarn add cfw-utils
```

Alternatively, you can also just copy and paste the utilities in [src/utils](src/utils) to your project.
